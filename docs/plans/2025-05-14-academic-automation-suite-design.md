# WordAI 学术自动化套件设计文档 (v2)

## 1. 概述
本套件旨在为 WordAI 增加深度学术排版能力，包含“表格美化（三线表）”与“智能参考文献管家”两个核心模块，显著提升学术论文与报告的编写效率。

## 2. 模块一：表格美化引擎 (Table Beautifier)
### 2.1 功能特性
- **一键三线表**：默认应用 1.5pt/0.75pt 标准学术线条。
- **智能复合表头**：识别跨列合并单元格，自动在次级表头下方添加栏目线。
- **批量扫描**：侧边栏列出全文表格，用户勾选后一键批量美化。
- **参数微调**：支持线条粗细、单元格边距、数字对齐方式的自定义。

## 3. 模块二：智能参考文献管家 (Reference Manager)
### 3.1 功能特性
- **占位符识别**：自动扫描文中 `【作者, 年份】`、`((...))` 等草稿占位符。
- **分步导航匹配**：针对不确定的引用，侧边栏提供“上一个/下一个”导航，并展示匹配建议供用户点击确认。
- **本地重格式化**：解析文末参考文献条目，按选定标准（GB/T, APA 等）重新排列要素，无需外部 API。
- **手动一键刷新**：支持全文编号（如 [1], [2]）及其超链接的一键更新。

## 4. 模块三：术语一致性检查 (Terminology Consistency)
### 4.1 功能特性
- **语义术语提取**：利用 LLM 自动提取文档中的核心学术术语集。
- **冲突扫描**：识别同一个概念在不同章节使用的不同表达（如“卷积神经网络”与“卷积感知机”）。
- **一键统一**：用户选择一个“标准术语”，插件自动替换全文中所有非标术语。

## 5. 模块四：智能图表编号同步 (Smart Numbering Sync)
### 5.1 功能特性
- **自动重排**：一键扫描全文“图 X”、“表 X”标题，按照出现顺序自动更新编号。
- **正文联动**：同步修改正文中所有对应的交叉引用文字（如“见图 3”自动改为“见图 4”）。
- **逻辑校验**：发现引用了不存在的图表编号时主动标红提醒。

## 6. 模块五：多合一摘要生成器 (Abstract Generator)
### 6.1 功能特性
- **全篇语义压缩**：AI 扫描全文，自动提炼“研究背景”、“核心方法”、“实验结果”、“结论”四个核心维度。
- **结构化摘要生成**：按照学术期刊标准格式自动合成摘要草稿。
- **关键词提取**：根据文章核心贡献点，自动生成 3-5 个高频/专业关键词。

## 7. 技术方案
- **基础架构**：使用 Office.js API 操作 `Word.Table`、`Word.Range` 和 `Word.Search`。
- **语义处理**：利用 LLM (OpenAI/Claude API) 进行长文档摘要、术语提取与逻辑仲裁。
- **状态维护**：利用 `BindingID` 或 `ContentControl` 追踪图表对象的位置。

## 8. 交互设计 (UI)
- **侧边栏 Tab 切换**：在 WordAI 原有功能旁增加“学术工具”标签。
- **极简紧凑风格**：圆角 4px，间距 8-12px，卡片化布局。
- **分步向导**：参考文献与术语检查采用“查找替换”式的交互逻辑。
