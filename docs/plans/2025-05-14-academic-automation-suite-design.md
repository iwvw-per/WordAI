# WordAI 学术自动化套件设计文档 (v2)

## 1. 概述
本套件旨在为 WordAI 增加深度学术排版能力，包含“表格美化（三线表）”与“智能参考文献管家”两个核心模块，显著提升学术论文与报告的编写效率。

## 2. 模块一：表格美化引擎 (Table Beautifier)
### 2.1 功能特性
- **一键三线表**：默认应用 1.5pt/0.75pt 标准学术线条。
- **智能复合表头**：识别跨列合并单元格，自动在次级表头下方添加栏目线。
- **批量扫描**：侧边栏列出全文表格，用户勾选后一键批量美化。
- **参数微调**：支持线条粗细、单元格边距、数字对齐方式的自定义。

## 3. 模块二：智能参考文献管家 (Reference Manager)
### 3.1 功能特性
- **占位符识别**：自动扫描文中 `【作者, 年份】`、`((...))` 等草稿占位符。
- **分步导航匹配**：针对不确定的引用，侧边栏提供“上一个/下一个”导航，并展示匹配建议供用户点击确认。
- **本地重格式化**：解析文末参考文献条目，按选定标准（GB/T, APA 等）重新排列要素，无需外部 API。
- **手动一键刷新**：支持全文编号（如 [1], [2]）及其超链接的一键更新。

## 4. 技术方案
- **基础架构**：使用 Office.js API 操作 `Word.Table` 和 `Word.Range`。
- **匹配算法**：基于正则与语义相似度的本地匹配逻辑。
- **链接实现**：采用“书签 + 超链接”或“REF 域 (OOXML)”实现文中到文末的跳转。
- **存储管理**：用户偏好与自定义模板存储于 `localStorage`。

## 5. 交互设计 (UI)
- **侧边栏 Tab 切换**：在 WordAI 原有功能旁增加“学术工具”标签。
- **分步向导**：参考文献匹配时采用“查找替换”式的交互逻辑。
