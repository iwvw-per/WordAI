# WordAI 表格美化（三线表）设计文档

## 1. 概述
为 WordAI 增加“一键三线表”功能，旨在帮助学术用户快速、智能地将普通表格转化为符合学术规范的三线表，并提供深度微调与全文档批量处理能力。

## 2. 核心功能
### 2.1 智能识别美化
- **智能表头检测**：自动识别复合表头（跨列合并单元格），并添加对应的次级栏目线。
- **结构化排版**：自动区分表头、正文和表注，应用差异化的线条和对齐方式。
- **内容自适应对齐**：文字默认左对齐，数值/百分比自动小数点或右对齐。

### 2.2 深度微调面板
- **线条控制**：支持自定义顶线、底线、栏目线的磅值（pt）。
- **布局微调**：支持调整单元格上下边距（Row Padding）以获得更好的视觉“呼吸感”。
- **模板切换**：预设国标（GB）、APA、Nature/Science 等多种学术风格，支持用户保存自定义配置。

### 2.3 批量处理扫描
- **全文扫描**：自动识别全文档所有表格并生成带标识的列表。
- **勾选确认**：用户在列表（Sidepane）中勾选需要处理的表格，避免误操作。
- **快速定位**：点击列表项，文档自动跳转至对应表格位置。

## 3. 技术方案
- **引擎**：基于 Office.js API (`Word.Table`)。
- **算法**：启发式逻辑判断表头层级与数据类型。
- **存储**：用户自定义模板保存在 `localStorage` 中。

## 4. 成功标准
- 能够准确识别并处理包含 2-3 层复合表头的表格。
- 批量处理 50+ 表格的文档时，侧边栏不卡顿且处理准确。
- 格式保留：美化过程不丢失表格内的公式、批注和交叉引用。
